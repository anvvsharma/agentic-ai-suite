# Learning Path

🎓 Structured learning path for Agentic AI & GenAI development.

## 📚 Phases Overview

### Phase 0: Prerequisites (1-2 weeks)
Foundation skills for AI development.

- **Pydantic** - Data validation and type safety
- **Python Advanced** - Decorators, context managers, async
- **Async Programming** - Async/await patterns

### Phase 1: LLM Fundamentals (2-3 weeks)
Core LLM concepts and usage.

- **LangChain Basics** - Chains, prompts, memory
- **Prompt Engineering** - Effective prompting techniques
- **LLM APIs** - OpenAI, Groq, Anthropic integration

### Phase 2: Embeddings & Vectors (2-3 weeks)
Vector operations and semantic search.

- **Vector Basics** - Embeddings and similarity
- **Similarity Search** - Cosine, euclidean metrics
- **Vector Databases** - ChromaDB, Pinecone, Weaviate

### Phase 3: RAG Systems (3-4 weeks)
Retrieval-Augmented Generation.

- **Document Processing** - Loading and parsing
- **Chunking Strategies** - Optimal text splitting
- **Retrieval Methods** - Dense, sparse, hybrid

### Phase 4: Agents Fundamentals (3-4 weeks)
Building autonomous agents.

- **Agent Architecture** - Design patterns
- **Reasoning Loops** - ReAct, Plan-Execute
- **Memory Systems** - Short-term, long-term memory

### Phase 5: Tool Calling (2-3 weeks)
Integrating external tools.

- **Function Calling** - LLM function calling
- **Tool Integration** - API integration patterns

### Phase 6: Multi-Agent Systems (3-4 weeks)
Coordinating multiple agents.

- **Agent Communication** - Message passing
- **Coordination Patterns** - Task delegation

## 🚀 Learning Approach

1. Follow phases sequentially (00 → 06)
2. Complete exercises in each phase
3. Build mini-projects to apply concepts
4. Move successful projects to finalized-pocs/
